# Student Performance Analysis

Statistical analysis and machine learning project developed in R using the Student Performance dataset from the UCI Machine Learning Repository.

## Objective

Identify the factors most associated with student performance through exploratory data analysis and classification models.

## Dataset

- UCI Machine Learning Repository
- Student Performance Dataset

## Tools

- R
- C5.0
- ggplot2
- Quarto

## Techniques

- Exploratory Data Analysis (EDA)
- Decision Trees
- Data Visualization
- Classification
